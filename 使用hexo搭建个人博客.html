使用hexo搭建个人博客

2021-03-30- 2021-03-31
- -

简介

这是一篇教你如何用hexo+github部署属于自己的博客简单教程

hexo博客在本地上运行

下载安装node.js

搭建博客首先需要安装node.js
node.js官方地址: https://nodejs.org/en/
建议下载长期支持版也就是左边的LTS

node.js是搭载hexo博客必备的东西

进入cmd

不推荐使用(简单):

使用win+R,进入到这个界面输入cmd，即可进入

推荐使用:

进入到自己想要存入的磁盘当中

例如,我进入的是E盘:

双击，然后输入cmd，一样可以进入cmd界面

搭建博客

已经成功进入cmd的话，我们可以输入node -v和npm -v来查看我们有没有安装好node.js

下载淘宝镜像:npm install -g cnpm --registry=https://registry.npm.taobao.org

官网也有具体步骤:https://hexo.io/zh-cn/

下载博客框架:cnpm install hexo-cli -g

创建博客空目录:mkdir blog

我们的博客都是会放到这个blog目录里，如果错了的话可以直接删掉blog目录重来

输入cd blog进入blog目录中，再输入hexo init初始化博客

初始化完成后，输入hexo s即可在本地端口中启动,成功则展示以下界面

hexo的基本用法

新建

我们可以用hexo n 你的博客标题来创建新的博客文章(我们创建的博客文章都会放在blog/source/_posts里)
博客文章用的是markdown语法,https://www.jianshu.com/p/191d1e21f7ed 这个链接是markdown里的一些语法

清除&更新

每次创建或者修改了博客都需要hexo cl和hexo g命令(这两条命令需要在cmd里blog目录下输入)

启动

要让博客在本地运行需要输入hexo s命令，即可在本地运行

上传

让博客上传到github，需要输入hexo d命令

当然我们搭建博客肯定是不可能只放在本地运行的，所以接下来教大家如何部署到github上，用github的地址访问我们的博客
